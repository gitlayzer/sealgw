4/7

4/7 层混合代理

底层基于高性能 LVS 实现四层代理，上层提供强大的七层路由、认证和 API 治理能力。

SEC

极致安全防护

原生支持 WAF、DDoS 缓解、TLS/SSL 强制终止，网关层为应用构建第一道防线。

API

统一 API 治理

提供动态限流、熔断和请求重写能力，对 Sealos 上的所有服务进行精细化控制。

HA

高可用设计

深度集成 `lvscare` 机制，确保网关入口和后端 Real Server 列表的持续高可用。

Sealgw 设计理念

性能优先

从底层 LVS 选型到上层代理实现，一切设计都以提供最小延迟和最大吞吐量为目标。

云原生管理

配置完全声明式，通过 Kubernetes API 管理

生态集成

与 Sealos 平台内的存储、监控、认证等模块深度融合，减少运维复杂性。
